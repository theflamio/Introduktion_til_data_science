1 Model selection

A model selection method consists of two components:

  1. A criterion for choosing one model over another

  2. A strategy for moving through possible candidate models (searching all possible models is usually impossible)

1.1 The curse of dimensionality

Previously, we specified a set of candidate models to choose among.

Very often, we have a family of models to choose among. Example: Multiple regression with \(p\) possible explanatory variables \(x_1,\dots, x_p\).

This is a very large family:

Either a predictor is included in the model - or it is not. This gives a total of \(2^p\) models to consider:

c(2^6, 2^16, 2^24, 2^31)
## [1] 6.40e+01 6.55e+04 1.68e+07 2.15e+09

If we allow for any degree of interaction the number of possible models is \(2^{p \choose 2}\) where \({p \choose 2}=p! /(2!(p-2)!)=p(p-1)/2\).

c(2^choose(6, 2), 2^choose(16, 2), 2^choose(24, 2), 2^choose(31, 2))
## [1]  3.28e+04  1.33e+36  1.21e+83 9.53e+139

The number \(2^\choose(24, 2)\) is in the vicinity of the estimated number of atoms in the universe.

1.2 A pragmatic strategy

Use AIC as selection criterion

Search in model space in stepwise way, e.g.:

  • Start with large model; remove terms trying to minimize AIC

  • Start with small model; add terms trying to minimize AIC

  • Start with some model; alternated between removing and adding terms trying to minimize AIC

Note:

  • In all cases we typically get local minimum of AIC

  • Models are often more or less meaningless when it comes to interpretation - but they may produce good predictions.

dat <- doBy::carcassall |> select(-slhouse)
set.seed(2024)
i <- sample(nrow(dat), size=0.1*nrow(dat))
train <- dat[i,]
test <- dat[-i,]
dat_num <- dat |> select_if(is.numeric)
round(10*cor(dat_num)) |> ggcorrplot(, hc.order = TRUE, type = "lower", lab = TRUE)

pp <- dat |> response_plot(LeanMeat ~ ., geoms = c(geom_jitter()))

lm_full <- lm(LeanMeat ~ ., data = train)
lm_1 <- lm(LeanMeat ~ 1, data = train)
lm_full |> AIC()
## [1] 140
lm_1 |> AIC()
## [1] 187
lm_forw <- step(lm_1, direction="forward", scope=terms(lm_full), trace=0)
lm_back <- step(lm_full, direction="backward", trace=0)
lm_both <- step(lm_1, direction="both", scope=terms(lm_full), trace=0)
model_list <- list(lm_1=lm_1, lm_forw=lm_forw, lm_both=lm_both, lm_back=lm_back, lm_full=lm_full)
model_list |> sapply(AIC)
##    lm_1 lm_forw lm_both lm_back lm_full 
##     187     133     131     126     140
model_list |> sapply(function(x) rmse(x, train))
##    lm_1 lm_forw lm_both lm_back lm_full 
##    3.59    1.34    1.35    1.15    1.11
model_list |> sapply(function(x) rmse(x, test))
##    lm_1 lm_forw lm_both lm_back lm_full 
##    3.64    2.22    2.22    2.22    2.26
